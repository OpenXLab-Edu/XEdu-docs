# XEduLLM安装

### 安装及更新方法

你可以通过pip命令来安装XEduLLM。

`pip install xedu-python` 或 `pip install XEdu-python`

更新库文件：

```python
pip install --upgrade XEdu-python
```

XEdu-python已经内置在XEdu的一键安装包中，解压后即可使用。

### 安装常见问题解答

1.系统原因

强烈建议win10及以上，win7不推荐。

2.依赖库问题

常见出问题的是onnxruntime库，建议手动pip安装，再进行库排查。
