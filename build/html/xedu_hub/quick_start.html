快速体验XEdu-Hub

简介

XEdu-Hub是一个集合了各种各样深度学习工具的模块，可以让用户高效地完成深度学习任务。

安装

pip instal XEdu-python或pip install xedu-python

体验

体验XEdu-hub的最快速方式是通过OpenInnoLab平台。

OpenInnoLab平台为上海人工智能实验室推出的青少年AI学习平台，满足青少年的AI学习和创作需求，支持在线编程。在“项目”中查看更多，查找“workflow”即可找到所有XEdu-hub相关的体验项目。

非正式demo：https://www.openinnolab.org.cn/pjlab/project?id=65083f0ae8de1d02739c9192&backpath=/pjedu/userprofile?slideKey=project&type=EDITOR#public

（用Chorme浏览器打开效果最佳）
